Readme

git console commands:
---------------------
md supercar-care-center
cd supercar-care-center
git clone https://github.com/Dimension-D/supercar-care-center.git .
git config --local user.name "<USERNAME>"
git config --local user.email "<EMAIL>"

git pull
git push

application run commands:
-------------------------
npm i
npm run dev

application build commands:
---------------------------
npm i
npm run build

npm install --package-lock-only
npm ci --cache .npm --prefer-offline

run the above 2 commands as well before checking in the code.
